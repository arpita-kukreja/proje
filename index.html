📊 AskYourData – AI-Powered Data Analysis Platform

Natural Language Analytics & Intelligent Data Visualization System


AskYourData is an advanced AI-driven analytics platform that enables users to upload CSV or Excel datasets and interact with them using plain English questions instead of SQL queries or coding. The application automatically understands dataset structures, generates executable Pandas code using LLMs, and provides meaningful answers with interactive charts and visual insights.


🎯 Key Features:

  • • Natural language querying for dataset analysis without coding
  • • AI-generated Python/Pandas code execution for dynamic analytics
  • • Automatic dataset insights generation on every file upload
  • • Interactive Plotly charts including bar, pie, line, and scatter graphs
  • • Conversation memory for context-aware follow-up questions
  • • Intelligent handling of large datasets with automatic sampling
  • • Missing value preprocessing and smart data preparation pipeline
  • • Self-correcting AI workflow with automatic retry on execution failure
  • • Session-based API rate limiting with live usage tracking
  • • Modern responsive dark-themed user interface

⚡ Technical Implementation:

Built using React for a high-performance frontend with interactive UI rendering. The backend is developed using FastAPI and Python 3.11 for efficient API handling and data processing. Google Gemini 2.5 Flash integrated through LangChain is used for intelligent code generation and reasoning. Pandas and OpenPyXL handle structured data processing, while Plotly.js provides dynamic visualizations. The system securely executes AI-generated analysis code and includes an automatic error-feedback retry mechanism for improved reliability. The frontend is deployed on Vercel, backend services on Render, and the entire platform is containerized using Docker.


🏗️ System Workflow:

Users upload CSV or Excel files, which are previewed client-side using PapaParse and SheetJS. The backend extracts schema information including column names, data types, sample rows, and statistical summaries. This metadata is sent to Gemini 2.5 Flash, which generates optimized Pandas and Plotly code dynamically. The generated code is executed against the dataset, and the resulting insights, textual responses, and chart configurations are returned to the frontend for visualization.


🌟 Project Impact:

AskYourData simplifies data analytics for non-technical users by removing the need for SQL, spreadsheets, or programming knowledge. The project demonstrates advanced full-stack engineering, AI integration, LLM orchestration, secure code execution, and interactive data visualization capabilities. It highlights expertise in building scalable AI-powered SaaS applications with real-world usability.
