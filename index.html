🔍 CodeLens – AI Powered GitHub Repository Reviewer

Advanced Developer Productivity AI Tool


CodeLens is an intelligent code review platform that leverages AI to analyze public GitHub repositories and deliver structured, actionable insights on code quality, scalability, and engineering best practices. It helps developers identify improvements faster and maintain high-quality code standards.


🎯 Key Features:

  • • Secure authentication with JWT
  • • GitHub repository analysis via REST API
  • • Insights on code quality, scalability, and best practices
  • • User-friendly web interface
  • • Structured feedback dashboard for developers

⚡ Technical Implementation:

Built using a MERN stack architecture with a Node.js and Express.js backend that integrates the GitHub REST API to fetch repository data. The system processes this data using Gemini API to generate intelligent insights. A React.js frontend visualizes the analysis in a clean, interactive dashboard, while MongoDB ensures efficient data handling.


🌟 Impact:

This project enhances developer productivity by automating code reviews and providing AI-driven insights, reducing manual effort and improving overall software quality. It showcases my ability to integrate AI into real-world developer tools.

🌐 Portfolio Builder – No-Code Portfolio Generator

Dynamic Web Development & Personal Branding Tool


Portfolio Builder is a full-stack platform that enables users to create, customize, and export professional portfolios without writing any code. It simplifies personal branding by offering real-time editing and instant deployment-ready portfolio generation.


🎯 Key Features:

  • • No-code portfolio creation with dynamic customization
  • • Real-time live preview of portfolio changes
  • • Secure JWT-based authentication system
  • • Export portfolios as standalone HTML files
  • • Responsive and modern UI using Tailwind CSS

⚡ Technical Implementation:

Developed using the MERN stack with React.js for a dynamic frontend and Node.js/Express.js backend for handling user data and authentication. MongoDB Atlas provides scalable cloud storage. The application supports real-time updates and is deployed using Vercel (frontend) and Render (backend).


🌟 Social Impact:

This project empowers users to build professional portfolios , eliminating technical barriers. It highlights my full-stack development skills, deployment knowledge, and ability to build user-centric, scalable applications.
